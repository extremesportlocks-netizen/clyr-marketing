  • Ozempic card says "From Novo Nordisk" + "The same proven GLP-1 medication trusted by millions worldwide" — Novo Nordisk attribution implies partnership; the "trusted by millions" is a passable softer claim but still risky.
  • Wegovy card likely has the same pattern (haven't fully audited).

Fix: regenerate in Stitch with text "Brand-name auto-injector" instead of "From [pharma co]," and a generic supporting line like "FDA-approved for weight loss" instead of the comparative claim. Or, edit text out post-export and we re-render in Photoshop/Figma.
